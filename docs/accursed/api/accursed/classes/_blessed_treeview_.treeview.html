Class TreeView<T>

A Tree widget made from scratch, this is inheriting directly from Element, and implementing render() without using any other Widget Implementation.

It doesn't support border, padding, or label. To fully support all features, wrap this element in a box.

It does support scrolling.

    whenever Box.render is called lpos gets set on the element, an object containing the rendered coordinates.
